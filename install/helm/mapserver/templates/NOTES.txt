No Ingress was created. Reach the services with port-forward:

  kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "mapserver.fullname" . }} 8888:{{ .Values.service.flaskPort }} 5001:{{ .Values.service.terracottaPort }}

Then open:
  Web UI:      http://localhost:8888
  Tile server: http://localhost:5001

For ollebo.com to reach this mapserver, you'll need to enable the Ingress
(--set ingress.enabled=true --set ingress.host=YOUR_DOMAIN) and set
terracotta.publicUrl to a publicly resolvable URL.
{{- end }}
